# Diagrams

The ASCII diagrams in `ARCHITECTURE.md` are the source of truth. PNG/SVG
exports go here when generated (e.g. via `mermaid-cli` or `excalidraw`).

Conventions:
- One diagram per file.
- Filename = a short slug, e.g. `data-flow-detection.svg`.
- Reference from docs as `![…](diagrams/data-flow-detection.svg)`.
